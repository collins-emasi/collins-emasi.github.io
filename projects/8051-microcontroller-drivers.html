8051 microcontroller driver development

Introduction

This Article contains code and documentation for projects that use the 8051 microcontroller in embedded systems. The 8051 is an old but versatile microcontroller that is still widely used in many applications.

In this project, I've developed drivers to interface the following peripherals

  • 7 Segment display
  • Bluetooth (simulation)
  • Counters and Timers
  • Keypad
  • LCD
  • RFID

And many others

In all these projects, I used Proteus Simulation software to test the execution of the firmware, but I'd be thrilled to see how it works on an actual 8051 MCU

All the code and information needed to get started and test the drivers are provided in my Github Account
