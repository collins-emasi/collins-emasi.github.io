Finding Amy rpoject

Description

Finding Amy project involved developing machine learning algorithms that will aid in the automation of classification of images taken by camera traps to help the Ecological Monitoring Department, EMD, in the Ol-Pejeta Conservancy better monitor the animals.

I was able to build a web-based application that an EMD user can upload images then they would be classified depending on the animal in the image and placed in separate folders.

In this project, I will describe the process of creating an image classification model for such an application

The notebook and information on how to get started can be found on my GitHub Profile
