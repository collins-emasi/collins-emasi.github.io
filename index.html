tinyRanger

  • Project : Embedded Systems
  • Technologies : Python, TensorflowLite, C/C++

Description

This is an open source environmental conservation project that I undertook as a part of my internship work at DSAIL. I leverage the power of tinyML to create a bird detection system which detects and classifys birds in an ecosystem. I then use LoRAWAN to send a flag to the cloud indicating that i have heard the bird spcies of interest. I then keep track of that in a dashboard and that is used to inform the best course of action to be taken in that ecosystem.

This project is still under continuous development and any possible colaboration is very much welcomed.
